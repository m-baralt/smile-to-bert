# Smile-to-bert

In this repository, a BERT trained to predict physicochemical properties from SMILES is provided. A BERT architecture with 4 encoder layers, 6 heads and an embedding size of 90 was trained to predict 7 physicochemical properties: number of H-bond acceptors, number of H-bond donors, number of rotatable bonds, exact mass, topological polar surface area (TPSA), number of heavy atoms, and log-P. The input consists of tokenized SMILES of a maximum length of 384.
